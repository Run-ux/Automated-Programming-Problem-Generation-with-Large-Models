### 题目标题：魔法阵中的元素波动

### 题目背景
在遥远的魔法王国中，有一座古老的魔法塔。这座塔内藏有无数神秘的魔法阵，每个魔法阵都由一系列元素晶石组成。这些晶石按照特定的顺序排列，能够产生强大的能量波动。年轻的法师们通过研究这些魔法阵，试图掌握其中蕴含的奥秘。

最近，一位年轻的法师发现了一种新的魔法阵，它由 $N$ 个元素晶石组成。每个晶石都有一个特定的能量值，用字符表示。法师发现，当这些晶石按照某种特定的顺序排列时，会产生一种特殊的能量波动，称为“美丽值”。美丽值的计算方法如下：
- 如果某个子序列的美丽值为 $k$，那么它的任意一个子序列的美丽值都不超过 $k$。
- 当前状态的美丽值可以通过前一个状态的美丽值和当前晶石的能量值来确定。

现在，这位年轻的法师想要知道，对于给定的魔法阵（即字符串），有多少种不同的排列方式可以使得整个魔法阵的美丽值不超过 $K$。

### 输入格式
输入包含两行：
- 第一行是一个整数 $N$ ($1 \leq N \leq 1000$)，表示魔法阵中元素晶石的数量。
- 第二行是一个长度为 $N$ 的字符串 $S$，表示魔法阵中每个元素晶石的能量值。

### 输出格式
输出一个整数，表示满足条件的不同排列方式的数量。结果对 $10^9 + 7$ 取模。

### 数据范围
- $1 \leq N \leq 1000$
- $0 \leq K \leq N$
- 字符串 $S$ 中的字符均为小写字母
- 时间限制：1.0s
- 内存限制：256MB

### 示例
#### 输入
```
3
abc
```

#### 输出
```
6
```

### 解释
对于字符串 "abc"，所有可能的排列方式有 "abc", "acb", "bac", "bca", "cab", "cba"。每种排列方式的美丽值都不超过 $K = 0$，因此答案为 6。
