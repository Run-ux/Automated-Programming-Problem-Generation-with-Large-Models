### 标题：社团活动日程安排

### 题目背景
在阳光明媚的校园里，学生们正热衷于参与各种丰富多彩的社团活动。为了更好地管理时间，学校决定优化所有社团活动的日程安排。现在，每个社团都有一系列必须参加的会议或活动（用字符串表示），并且希望找到一种方式将这些活动尽可能紧凑地安排在一起，以减少总占用的时间。你的任务是帮助学校设计一个算法，使得所有社团的活动能够被安排在一个最短的时间段内，同时满足各个社团的需求。

### 问题描述
假设你是一名聪明的学生会成员，负责协调全校的社团活动安排。给定 $K$ 个社团各自需要完成的一系列活动（每个社团的活动列表由一系列字符串组成），你需要找到一种方法把这些活动串起来，形成一个“超级日程表”，这个日程表应该包含所有社团提供的所有活动，并且长度尽可能短。这里的“长度”指的是从第一个活动开始到最后一个活动结束所经过的时间总和。

### 输入格式
- 第一行是一个整数 $K (1 \leq K \leq 6)$, 表示社团的数量。
- 接下来有 $K$ 组输入，每组的第一行是一个整数 $n_i (1 \leq n_i \leq 100)$, 表示第 $i$ 个社团需要进行的活动数目。
- 紧接着是 $n_i$ 行，每行包含一个字符串 $s_{ij}$ $(1 \leq |s_{ij}| \leq 14311)$, 表示第 $j$ 项活动的名称。
- 总共的字符数不超过 $14311$。

### 输出格式
输出一个整数，代表能容纳所有社团活动的最短“超级日程表”的长度。

### 数据范围
- 时间限制: 1.0s
- 内存限制: 256MB
- $1 \leq K \leq 6$
- 对于每一个社团 $i$, $1 \leq n_i \leq 100$
- 每个活动的名字长度 $|s_{ij}| \leq 14311$

### 示例
**输入**
```
2
3
MathClub
ScienceLab
Robotics
2
DramaPlay
MusicBand
```

**输出**
```
18
```

**解释**
这里的一种可能解法是先安排 "MathClub" 和 "ScienceLab"，然后插入 "Robotics"，最后加上 "DramaPlay" 和 "MusicBand"，得到总长度为 18 的日程表。注意可能存在其他同样有效的排列方式。
