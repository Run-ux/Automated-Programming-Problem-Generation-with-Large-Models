# 摊位分区规划

> Variant: `CAMPUS_BOOTH_COUNT` / `v1` / theme `campus_ops`

## 题目描述

校园文化节即将开幕，学生会需要将一条直道上的 $n$ 个社团摊位划分为若干个连续的管理区。每个摊位有一个热度值 $a_i$。

对于任意一个划分出的连续区间 $[l, r]$，其管理成本计算规则如下：
- 若区间内所有摊位的热度值均能被该区间的最小热度值整除，则称该区间为“和谐区间”，其管理成本等于该区间的最小热度值。
- 否则，该区间为“松散区间”，其管理成本等于 $p \times (r - l)$，即区间内相邻摊位对的数量乘以基准协调开销 $p$。

整个划分方案的总成本为所有管理区成本之和。请计算使总成本最小的不同划分方案数。由于答案可能很大，请输出对 $10^9+7$ 取模的结果。

注意：两种划分方案不同，当且仅当存在至少一个切割位置不同。

## 输入格式

第一行包含一个整数 $t$，表示测试用例的数量。
每个测试用例的第一行包含两个整数 $n$ 和 $p$。
第二行包含 $n$ 个整数 $a_1, a_2, \dots, a_n$。

## 输出格式

对于每个测试用例，输出一行一个整数，表示最小总成本对应的划分方案数对 $10^9+7$ 取模的结果。

## 数据范围与限制

- $1 \le t \le 10^4$
- $2 \le n \le 2 \times 10^5$
- $1 \le p \le 10^9$
- $1 \le a_i \le 10^9$
- 所有测试用例的 $n$ 之和不超过 $2 \times 10^5$。
- 时间限制：2 秒
- 空间限制：256 MB

## 样例

### 样例 1

**输入**

```text
1
3 2
4 2 4
```

**输出**

```text
1
```

**说明**

样例1：最小总成本为 2。唯一最优方案为不切割，将整个区间 [1,3] 作为一个和谐区间。该区间最小值为 2，且 2 能整除 4、2、4，因此成本为 2。其他任何切割方式（如 [1,1] 与 [2,3]）的总成本均大于 2。

### 样例 2

**输入**

```text
1
3 2
2 3 2
```

**输出**

```text
3
```

**说明**

样例2：最小总成本为 4。共有 3 种最优划分方案：
1. 不切割：区间 [1,3] 为松散区间（最小值 2 不能整除 3），成本为 2*(3-1)=4。
2. 在位置 1 后切割：区间 [1,1] 成本为 2；区间 [2,3] 为松散区间，成本为 2*(2-1)=2。总成本 4。
3. 在位置 2 后切割：区间 [1,2] 为松散区间，成本为 2；区间 [3,3] 成本为 2。总成本 4。
其他划分方式成本均大于 4，故答案为 3。

## 说明

方案数需对 $10^9+7$ 取模。划分方案由切割位置集合唯一确定，不同切割集合视为不同方案。单元素区间恒为和谐区间，其成本等于该元素的热度值。
