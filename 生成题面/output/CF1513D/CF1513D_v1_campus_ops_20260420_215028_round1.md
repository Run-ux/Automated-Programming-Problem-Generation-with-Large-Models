# 校园最优网络方案数

> Variant: `CAMPUS_OPTIMAL_NETWORK_COUNT` / `v1` / theme `campus_ops`

## 题目描述

某大学新校区有 n 间教室沿主干道一字排开，编号为 1 到 n。为了保障师生通行与物资调度，后勤部计划在教室之间修建走廊。每间教室 i 有一个基础维护系数 a_i。走廊的修建与成本核算遵循以下规则：
1. 标准走廊：任意相邻的两间教室 i 与 i+1 之间必定可以修建一条标准走廊，其维护成本固定为 p。
2. 直达走廊：对于任意两间教室 i 与 j（i < j），若区间 [i, j] 内所有教室的维护系数均能被该区间的最小值整除，则允许在 i 与 j 之间修建一条直达走廊，其维护成本等于该区间的最小值。
学校希望以最小的总维护成本将所有教室连通（即构成一棵生成树）。后勤部需要统计在总成本最小的前提下，共有多少种不同的走廊修建方案。两种方案被视为不同，当且仅当它们所选的走廊集合不完全相同。由于方案数可能极大，请输出结果对 10^9+7 取模的值。

## 输入格式

第一行包含一个正整数 t，表示测试用例的数量。
接下来 t 行，每行描述一个测试用例：
第一行包含两个正整数 n 和 p，分别表示教室数量和标准走廊的固定成本。
第二行包含 n 个正整数 a_1, a_2, ..., a_n，表示各教室的维护系数。

## 输出格式

对于每个测试用例，输出一行一个整数，表示最优连通方案数对 10^9+7 取模的结果。

## 数据范围与限制

- 1 ≤ t ≤ 10^4
- 2 ≤ n ≤ 2 × 10^5
- 1 ≤ p, a_i ≤ 10^9
- 所有测试用例的 n 之和不超过 2 × 10^5
- 时间限制：2.0 秒
- 空间限制：256 MB

## 样例

### 样例 1

**输入**

```text
2
3 100
2 2 2
4 10
2 4 8 16
```

**输出**

```text
3
16
```

**说明**

样例 1：所有合法直达走廊成本均为 2，低于标准走廊成本 100。在 3 个节点间任选 2 条成本为 2 的走廊即可构成最小生成树，共有 3 种方案。样例 2：所有合法直达走廊成本均为 2，图退化为边权全为 2 的完全图。4 个节点的完全图生成树数量为 16，均达到最小总成本。

### 样例 2

**输入**

```text
1
3 5
3 6 9
```

**输出**

```text
1
```

**说明**

样例 3：合法直达走廊仅有 (1,2) 成本 3 和 (1,3) 成本 3，(2,3) 不满足整除条件。标准走廊成本为 5。最小总成本为 6，唯一方案为选择 (1,2) 与 (1,3)。

## 说明

本题要求计算最小总成本下的不同连通方案数。答案需对 10^9+7 取模。若存在多种方案达到相同最小成本，需全部统计。数据保证图始终连通。
