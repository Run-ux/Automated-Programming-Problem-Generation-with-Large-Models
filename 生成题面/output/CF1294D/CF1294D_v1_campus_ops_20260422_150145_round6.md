# 校园时段调度

> Variant: `CAMPUS_COVER_CERT` / `v1` / theme `campus_ops`

## 题目描述

校园活动中心的排课系统需要处理社团队伍的教室借用申请。时间轴被抽象为无限延伸的非负整数序列 0, 1, 2, ...，每个整数代表一个独立的时段。系统以周期 x 循环划分时段属性。

每支队伍 i 按顺序抵达时，会提交一个基础偏好时间 a_i。由于排课弹性，该队伍可以被安排到任意时段 s，当且仅当 s ≡ a_i (mod x) 或 s ≡ a_i + 1 (mod x)。

教务系统会严格按输入顺序依次处理两类操作：
1. 新队伍抵达，记录其偏好时间。队伍编号按抵达顺序从 1 开始递增。
2. 发起一次覆盖查询：给定正整数 K，要求判断能否从当前已抵达的队伍中选出恰好 K 支，为时段 0, 1, ..., K-1 各分配一支不同的队伍（即完美覆盖）。

若存在合法分配方案，请输出具体方案；若不存在，根据 Hall 定理，必然存在某个子区间 [L, R]（0 ≤ L ≤ R < K），其能覆盖的队伍总数严格小于区间长度 R - L + 1。此时需输出字典序最小的冲突区间 [L, R]。

注：队伍 i 能覆盖区间 [L, R]，当且仅当其合法时段集合 {a_i mod x, (a_i+1) mod x} 与 [L, R] 存在交集。

## 输入格式

第一行包含两个正整数 x 和 q，分别表示时段周期长度与操作总数。
接下来 q 行，每行描述一次操作，按时间顺序依次给出：
- 1 a_i：表示一支新队伍抵达，其基础偏好时间为 a_i。
- 2 K：表示一次覆盖查询，要求判断能否覆盖时段 0 至 K-1。

## 输出格式

对于每次 2 K 查询，输出一组结果：
- 若存在合法分配方案，第一行输出 YES。随后 K 行，每行输出两个整数 id s，表示将编号为 id 的队伍分配至时段 s。任意合法方案均可，其中 s 必须满足 0 ≤ s < K。
- 若不存在合法方案，第一行输出 NO。第二行输出两个整数 L R，表示字典序最小的冲突区间（即优先最小化 L，其次最小化 R）。

## 数据范围与限制

- 1 ≤ x, q ≤ 4×10^5
- 0 ≤ a_i ≤ 10^9
- 1 ≤ K ≤ 10^9
- 时间限制：2.0 秒
- 空间限制：512 MB
- 保证所有输出 YES 的查询中 K 的总和不超过 4×10^5

## 样例

### 样例 1

**输入**

```text
3 4
1 0
1 1
2 2
2 3
```

**输出**

```text
YES
1 0
2 1
NO
0 2
```

**说明**

周期 x=3。队伍1（a=0）合法时段为 {s | s ≡ 0, 1 (mod 3)}。队伍2（a=1）合法时段为 {s | s ≡ 1, 2 (mod 3)}。
查询 K=2，目标区间 [0, 1]。两支队伍均能覆盖该区间（交集非空），且队伍数 2 ≥ 2。可分配队伍1至时段0，队伍2至时段1，满足约束。
查询 K=3，目标区间 [0, 2]。当前仅有2支队伍，总数 2 < 3，必然无解。检查子区间发现 [0, 2] 本身覆盖队伍数为2，小于区间长度3，构成冲突。字典序最小的冲突区间为 [0, 2]。

### 样例 2

**输入**

```text
5 4
1 0
1 3
1 3
2 4
```

**输出**

```text
NO
0 1
```

**说明**

周期 x=5。队伍1（a=0）覆盖 {s | s ≡ 0, 1 (mod 5)}。队伍2、3（a=3）覆盖 {s | s ≡ 3, 4 (mod 5)}。
查询 K=4，目标区间 [0, 3]。
检查子区间 [0, 1]：仅队伍1的合法时段与该区间有交集，覆盖队伍数为1，小于区间长度2，违反条件。
检查子区间 [2, 2]：无任何队伍覆盖，覆盖数为0，小于长度1。
比较字典序，[0, 1] 的左端点更小，故输出 [0, 1]。

## 说明

1. 模运算定义：s ≡ a (mod x) 表示 s 除以 x 的余数等于 a 除以 x 的余数。时段编号为非负整数。
2. 字典序定义：区间 [L1, R1] 字典序小于 [L2, R2] 当且仅当 L1 < L2 或 L1 = L2 且 R1 < R2。
3. 覆盖判定：队伍 i 能覆盖区间 [L, R] 当且仅当其合法时段集合与 [L, R] 存在交集。冲突区间 [L, R] 需满足覆盖该区间的有效队伍总数 < R - L + 1。
4. 方案输出：当输出 YES 时，任意满足约束的分配方案均被接受，时段 s 必须严格落在查询区间 [0, K-1] 内。
5. 数据规模提示：由于 K 可能极大，直接枚举区间不可行。请利用前缀盈余性质与高效数据结构定位极小违反集。
