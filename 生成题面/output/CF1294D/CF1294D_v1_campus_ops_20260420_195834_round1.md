# 校园WiFi信道鲁棒分配

> Variant: `CampusWiFi_RobustMEX` / `v1` / theme `campus_ops`

## 题目描述

校园网管中心近期需要为 n 个学生社团集中分配 WiFi 信道。每个社团提交了一个基准请求频率 a_i。由于硬件滤波器的物理限制，实际分配给社团的信道频率必须与基准请求频率模 x 同余，且分配值必须为非负整数。

近期校园网络出现周期性抖动，导致各余数类 r（0 ≤ r < x）的实际可用设备数量 c_r' 可能偏离网管中心统计的名义数量 c_r。已知抖动造成的总偏移量受预算 Δ 限制，即所有余数类的实际数量与名义数量之差的绝对值总和不超过 Δ（∑_{r=0}^{x-1} |c_r' - c_r| ≤ Δ），且实际数量 c_r' 保持非负。

网管中心希望制定一个鲁棒的分配预案：寻找最大的整数 K，使得无论网络抖动如何分配（只要满足上述预算 Δ 约束），中心都能找到一种合法的设备分配方案，使得频率 0, 1, 2, ..., K-1 均被恰好一个社团占用。请计算这个最大的 K 值。

## 输入格式

第一行包含三个整数 n, x, Δ，分别表示社团数量、信道间隔模数和扰动预算。
第二行包含 n 个整数 a_1, a_2, ..., a_n，表示各社团的基准请求频率。

## 输出格式

输出一个整数，表示满足条件的最大保底阈值 K。

## 数据范围与限制

- 1 ≤ n, x ≤ 200000
- 0 ≤ Δ ≤ 10^9
- 0 ≤ a_i ≤ 10^9
- 时间限制：2.0 秒
- 空间限制：512 MB

## 样例

### 样例 1

**输入**

```text
5 3 1
0 3 1 4 2
```

**输出**

```text
2
```

**说明**

名义统计中，模 3 余 0 的设备有 2 个（0, 3），余 1 的有 2 个（1, 4），余 2 的有 1 个（2）。扰动预算 Δ=1。若尝试 K=3，需要各余数类至少 1 个设备。最坏情况下，抖动可能将余 2 类的 1 个设备全部削减（偏移量 1 ≤ Δ），导致余 2 类实际可用为 0，无法覆盖频率 2，故 K=3 不成立。对于 K=2，只需覆盖频率 0 和 1，分别需要余 0 和余 1 类各 1 个设备。即使抖动削减任意一类 1 个设备，剩余名义数量仍满足需求（2-1=1 ≥ 1）。因此最大保底 K 为 2。

### 样例 2

**输入**

```text
4 2 0
0 0 1 1
```

**输出**

```text
4
```

**说明**

扰动预算 Δ=0，表示无抖动，实际数量等于名义数量。模 2 余 0 有 2 个，余 1 有 2 个。覆盖 0,1,2,3 恰好需要余 0 类 2 个（对应 0,2），余 1 类 2 个（对应 1,3）。名义数量完全匹配，故 K=4 成立。

## 说明

本题要求的是最坏情况下的保底阈值，即对于所有满足 ∑|c_r' - c_r| ≤ Δ 的合法扰动分布，都必须存在覆盖 [0, K-1] 的分配方案。若不存在任何正整数 K 满足条件，请输出 0。所有运算均在整数范围内进行，无需取模。
