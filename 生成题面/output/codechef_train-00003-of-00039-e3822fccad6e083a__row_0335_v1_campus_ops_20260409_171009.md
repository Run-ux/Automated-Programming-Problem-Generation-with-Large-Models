# 社团招新布线

> Variant: `campus_ops_counting_001` / `v1` / theme `campus_ops`

## 题目描述

新学期伊始，学校主干道旁规划了一排编号为 1 到 N 的连续摊位，用于各社团招新。为了保障用电安全，走廊两端（位置 0 和位置 N+1）已预先架设了电源塔。各社团将按某种顺序依次入驻摊位。每次入驻时，该社团必须从当前尚未使用的空位中任选一个，并拉设一条电线连接到距离该空位最近的已通电位置（可以是两端的电源塔，也可以是已经入驻并通电的社团摊位）。电线的长度即为两个位置编号之差的绝对值。入驻完成后，该摊位自动通电，后续社团可将其作为连接点。所有 N 个社团入驻完毕后，统计拉设电线的总长度。现在，学校后勤处希望知道：有多少种不同的社团入驻顺序（即 1 到 N 的排列），能够使得最终拉设的电线总长度不超过给定的预算 M？由于方案数可能极大，请输出结果对给定质数 P 取模后的值。

## 输入格式

第一行包含一个正整数 T，表示测试用例的数量。
接下来 T 行，每行包含三个正整数 N, M, P，分别表示摊位数量、预算上限以及取模用的质数。

## 输出格式

对于每组测试用例，输出一行一个整数，表示满足总长度不超过 M 的入驻排列总数对 P 取模的结果。

## 数据范围与限制

- 1 ≤ T ≤ 10
- 1 ≤ N ≤ 100
- 1 ≤ M ≤ 5000
- 10^8 ≤ P ≤ 10^9+7，且 P 为质数
- 时间限制：1.0 秒
- 空间限制：256 MB

## 样例

### 样例 1

**输入**

```text
2
1 1 1000000007
2 2 1000000007
```

**输出**

```text
1
2
```

**说明**

第一组：N=1，仅有一个摊位（位置1）。入驻时距离最近电源塔（0或2）均为1，总长度必为1。预算M=1，满足条件的排列只有[1]，共1种。
第二组：N=2，摊位为1和2。排列[1,2]：先入驻1，连0长度1；再入驻2，连1长度1；总长2。排列[2,1]：先入驻2，连3长度1；再入驻1，连2长度1；总长2。两种排列总长均为2，均不超过预算M=2，故答案为2。

### 样例 2

**输入**

```text
1
3 3 1000000007
```

**输出**

```text
2
```

**说明**

N=3时，总长度可能为3或4。经枚举，仅有排列[1,2,3]和[3,2,1]的总长度恰好为3，其余4种排列总长度均为4。预算M=3，故满足条件的排列共2种。

## 说明

本题要求精确计数并对输入的质数 P 取模。若不存在任何排列满足总长度约束，请输出 0。排列顺序不同即视为不同方案（例如 [1,2] 与 [2,1] 算作两种）。
