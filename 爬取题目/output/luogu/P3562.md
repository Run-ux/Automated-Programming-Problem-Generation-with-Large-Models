[POI 2013] LAS-Laser

平面上有些线段，你最多从原点射出 $k$ 条射线，穿过最多的线段，且使得每条线段最多被穿过 $1$ 次。

求最多能穿过多少线段。

Input
第一行两个整数 $k$，$n$，其中 $n$ 是线段数量。  

之后 $n$ 行，每行四个整数 $x_1,y_1,x_2,y_2$ 表示一条线段。

Output
输出一个整数，表示最多能穿过几条线段。

Constraints
对于 $100\%$ 的数据，$1 \leq k\le100$，$1 \leq n\le5\times10^5$，$1 \leq x_1, y_1, x_2, y_2 \leq 10^5$。
