[POI 2001 R1 / ZJOI2006 / HAOI2007] 反素数

对于任何正整数 $x$，其约数的个数记作 $g(x)$。例如 $g(1)=1$，$g(6)=4$。

如果某个正整数 $x$ 满足：$\forall 0 \lt i \lt x$，都有 $g(x) \gt g(i)$，则称 $x$ 为**反素数**。例如，$1,2,4,6,12,24$ 等都是反素数。

现在给定一个正整数 $N$，你能求出不超过 $N$ 的最大的反素数么？

Input
仅一行一个正整数 $N$。

Output
仅一行一个正整数，代表不超过 $N$ 的最大的反素数。

Constraints
对于所有数据，有 $1 \leq N \leq 2 \times 10^9$。
