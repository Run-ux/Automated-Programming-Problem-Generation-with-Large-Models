[蓝桥杯 2023 省 C] 异或和之差

给定一个含有 $n$ 个元素的数组 $A_i$，你可以选择两个不相交的子段。求出这两个子段内的数的异或和的差值的最大值。

Input
输入的第一行包含一个整数 $n$。

第二行包含 $n$ 个整数 $A_i$，相邻整数之间使用一个空格分隔。

Output
输出一行包含一个整数表示答案。

Constraints
两个子段可以分别选 $1$ 和 $4, 9, 2$，差值为 $15 - 1 = 14$。
对于 $40\%$ 的评测用例，$n \leq 5000$；
对于所有评测用例，$2 \leq n \leq 2 \times 10^5$，$0 \leq A_i \leq 2^{20}$。
