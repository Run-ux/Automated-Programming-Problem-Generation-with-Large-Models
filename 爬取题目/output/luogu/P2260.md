[清华集训 2012] 模积和

数学题，无背景。

求

$$\sum_{i=1}^{n} \sum_{j=1}^{m} (n \bmod i) \times (m \bmod j), i \neq j$$

$\bmod 19940417$ 的值

Input
输入只有一行两个整数 $n$，$m$。

Output
答案 $\bmod 19940417$

Constraints
#### 数据规模与约定
- 对于 $10\%$ 的数据，保证 $n,m \leq 10^3$。
- 对于 $30\%$ 的数据，保证 $n,m \leq 10^6$。
- 另有 $30\%$ 的数据，保证 $n \leq 100$。
- 对于 $100\%$ 的数据，保证 $1 \leq n,m \leq 10^9$。
