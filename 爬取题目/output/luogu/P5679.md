[GZOI2017] 等差子序列

GZOI2017 D2T2

给出一个长度为 $N$ 的正整数序列，问是否存在一个长度不小于三的等差子序列。

Input
第一行一个正整数 $T$，表示数据组数。

接下来 $T$ 组数据，每组数据第一行一个正整数 $N$，表示序列长度，接下来第二行有 $N$ 个正整数，依次表示序列中的数。

Output
对于每组数据，输出单独的一行 `YES` 或 `NO`。

Constraints
对于 $20\%$的数据：$N\le 100$。
对于 $40\%$的数据：$N\le 10^3$。
对于 $100\%$的数据：$1\le T \le 10$，$1\le N\le 2\times 10^4$，序列中的数 $\le 2\times 10^4$。
