C. Делимость на 2023

Даны две различные десятичные цифры $A$ и $B$.

Выведите любое целое положительное число $N$, составленное только из этих цифр (или одной из них), которое нацело делится на $2023$.

Input
В единственной строке даны через пробел два целых числа $A$ и $B$ $(0 \le A \lt B \le 9)$ — разрешенные десятичные цифры.

Output
Выведите целое положительное число $N$ $(1 \le N \lt 10^{100})$, которое нацело делится на $2023$; состоит только из цифр $A$ и $B$ (или одной из них). Число $N$ не должно иметь ведущих нулей (старшая цифра должна быть отлична от $0$). Если корректных ответов несколько — выведите любой из них.

Constraints
ограничение по времени на тест 1 секунда
ограничение по памяти на тест 256 мегабайт
