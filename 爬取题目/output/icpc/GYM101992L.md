L. Reflection

You are given a line with equation x = y. you need to perform the following operation Q times.

Note that each operation depends on all operations before it.

Input
The first line of the input is the number of test cases T, each test case starts with a line containing a single integer Q the number of operations, where 1 ≤ Q ≤ 10 5. Each of the following Q lines represents a query with a single integer x, where 0 ≤ x i ≤ 10 5.

Output
For each test case output Q lines. Each of them contains the corresponding y value.

Constraints
time limit per test 15 seconds
memory limit per test 1024 megabytes
