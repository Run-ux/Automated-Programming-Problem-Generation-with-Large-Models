Input
There are multiple test cases. The first line contains an integer $T$ ($1\le T\le 10^6$) indicating the number of test cases, for each test case: The first line contains an integer $n$ ($1\le n\le 10^6$) indicating the length of magic sequence. The second line contain $n$ integers $v_1, v_2, \ldots, v_n$ ($1\le v_i\le 10^6$), represents the magic sequence. It is guaranteed that the sum of $n$ in all data does not exceed $10^6$.

Output
For each test case, print a single integer, represents the maximum sum modulo $998244353$.

Constraints
time limit per test 3 seconds
memory limit per test 1024 megabytes
