A. AND-OR closure

Find the size of the AND-OR closure of $A$.

Here AND denotes the bitwise AND operation, and OR denotes the bitwise OR operation.

Input
The first line of the input contains a single integer $n$ ($1 \leq n \leq 2 \cdot 10^5$)  — the size of the set $A$. The second line contains $n$ distinct integers $a_1$, $a_2$, $\ldots$, $a_n$ ($0\leq a_i< 2^{40}$)  — which represent the elements of $A$.

Output
Print the size of the AND-OR closure of $A$.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
