I. Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly steps through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted.

You are asked to count the number of permutations of $1$ to $n$ which, after $k$ times passing through the list in the bubble sort, would become an almost sorted permutation.

Input
The input contains several test cases, and the first line is a positive $T$ indicating the number of test cases which is up to $5000$. For each test case, a line contains three integers $n$, $k~(1 \leq n, k \leq 50)$ which are described as above, and $q~(10^8 \leq q \leq 10^9)$ which is a prime number for the output.

Output
For each test case, output a line containing Case #x: y, where x is the test case number starting from $1$. And y is the remainder of the number of permutations which meet the requirement, dividing by $q$.
