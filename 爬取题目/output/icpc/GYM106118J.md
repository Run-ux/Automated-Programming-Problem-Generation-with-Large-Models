You are given a map with Doraemon's workshop and all the kids' houses. The roads connect these places, and each road has a certain length. Every house is either a "Good Kid" house or a "Bad Kid" house.

You are guaranteed that Santa can always find a way to reach every house by following these rules.

Input
The first line has one number, $n$ ($1 \le n \le 10^5$), which is how many houses there are. The second line has a string of $n$ letters. Each letter tells you the type of a house. 'G' is for a Good kid's house, and 'B' is for a Bad kid's house. The third line has one number, $m$ ($1 \le m \le 3 \cdot 10^5$), which is how many roads there are on the map. The next $m$ lines describe the roads. Each line has three numbers: $u$, $v$, and $d$ ($0 \le u,v \le n, 1 \le d \le 10^6$). This means there is a road between place $u$ and place $v$ with a distance of $d$. Place $0$ is Doraemon's workshop, and places $1$ to $n$ are the houses.

Output
Print one single number — the shortest total distance Santa must travel for all the deliveries combined.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
