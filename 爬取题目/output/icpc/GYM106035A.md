A. Pyramidal paths

Recently, in a class on discrete mathematics, student Sasha learned about pyramidal Hamiltonian paths. A Hamiltonian path is a path in a graph that passes through each vertex exactly once. A distinctive property of pyramidal Hamiltonian paths is that the vertex numbers in them either go first in ascending order of vertex numbers, and then in descending order of numbers, or first the numbers go in descending order of vertex numbers, and then in ascending order of numbers, and the intervals of ascending and descending must contain at least two elements (in this case, the intervals of ascending and descending may have a common element).

Input
The first line contains the number $n$ $(3 \le n \le 10^5)$ – the number of vertices in the Hamiltonian path. The second line contains $n$ pairs of different numbers $a_i$ $(1 \le a_i \le n)$ – enumeration of the vertices of the path.

Output
In a single line, print "YES" (without quotes) if the specified Hamiltonian path is pyramidal, and "NO" otherwise.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
