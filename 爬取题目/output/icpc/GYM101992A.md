A. Zeros and Ones

Given four integers x, y, M and K. You need to count the number of integers d that satisfy the following rules:

Input
The first line of the input is the number of test cases T. Each test case consists of four integers x, y, M and K, where 1 ≤ M ≤ 10 9, 1 ≤ x + y ≤ 42, x ≥ 1, y ≥ 0 and 0 ≤ K < M.

Output
For each test case output a single line with the number of integers d that satisfy the rules.

Constraints
time limit per test 15 seconds
memory limit per test 1024 megabytes
