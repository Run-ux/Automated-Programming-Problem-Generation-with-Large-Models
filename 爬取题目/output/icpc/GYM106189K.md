K. Factorials

Natural numbers are a fascinating subject of study, and mathematicians constantly confirm this by discovering more and more interesting patterns. Equally interesting are integer functions of natural numbers, such as factorials.

It is obvious that any natural $n$ can be represented as the ratio of two factorials: $$ n = \frac{n!}{(n - 1)!}.$$

It is also well known that some integers can be represented as the ratio of one factorial to the product of two others greater than 1: $$ n = \frac{a!}{b! \times c!}.$$

An example of such numbers are the binomial coefficients: $$ C_n^k = \frac{n!}{k! (n-k)!} \quad => \quad 6 = \frac{4!}{2! \times 2!}.$$

Let us limit ourselves to considering factorials up to $ 50\,000! $, and numbers up to $ 1\,000\,000\,000 $. Write a program that, given a number $n$, determines whether it can be represented as $$ n = \frac{a!}{b! \times c!}, $$ where $c \le b < a$.

Input
The input consists of a single integer $n$ ($1 \le n \le 10^9$).

Output
In a single line, print three integers $a$, $b$, and $c$ ($1 < c \le b < a \le 50\,000$), separated by spaces. If this representation is not the only one, print the one with the smallest numerator $ a $, and if there are several such solutions, then among them, select the one with the smallest number $ b $. If no solution exists within the constraints, print three numbers $-1$.

Constraints
time limit per test 3 seconds
memory limit per test 256 megabytes
