E. Pluses and minuses

The input to your program is a valid arithmetic expression that can contain only integers in the range from 0 to 1000, parentheses, and the signs of binary operations '+' and '-'. There are no unary operations.

You are allowed to change any plus sign to a minus sign, and any minus sign to a plus sign. Change the signs so that the value of the expression is as small as possible.

Input
A single line of input data contains an expression in the format described above, no more than $10^5$ characters long.

Output
Print the resulting expression after changing the characters. If there are multiple solutions, print any of them.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
