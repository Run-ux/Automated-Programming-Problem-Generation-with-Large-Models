E. Exotic Array

An array $a = [a_1,a_2,\dots, a_n]$ of length $n$ is exotic if and only if these two conditions are satisfied.

You are given two integers $n$ and $m$. Count the number of exotic arrays $a$ of length $n$. Since the answer can be very large, print it modulo $10^9 + 7$.

Input
The first line contains a single integer $t$ ($1 \le t \le 10^4$) — the number of test cases. Each of the next $t$ lines contains two integers $n$, $m$ ($1 \le n, m \le 10^9$) — the length of the array and the maximum allowed value.

Output
For each test case, print a single integer — the number of exotic arrays $a$ modulo $10^9 + 7$.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
