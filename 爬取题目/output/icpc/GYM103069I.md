Prof. Pang wants to know the death time (in seconds) of all the $n$ zombies.

Input
The first line contains five integers $n,m,V,K,D$ ($1\le n,m \le 10^5, 1\le V,K,D \le 10^9$) separated by single spaces. Each of the following $n$ lines contains two integers $t_i, h_i$ ($1\le t_i,h_i \le 10^9$) separated by a single space. Each of the following $m$ lines contains two integers $p_i, a_i$ ($1\le p_i,a_i \le 10^9$) separated by a single space.

Output
Output one line containing $n$ integers, where the $i$ -th integer denotes the death time (in seconds) of the $i$ -th zombie.

Constraints
time limit per test 2 seconds
memory limit per test 256 megabytes
