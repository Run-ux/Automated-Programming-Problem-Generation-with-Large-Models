J. Parallelograms

There are n sticks, the i -th of which has length a i. Alex wants to assemble from them as many parallelograms as possible simultaneously, with each stick used at most in one parallelogram. What maximal number of parallelograms is it possible to assemble?

Input
The first line contains a single integer n (1 ≤ n ≤ 200000) — the number of sticks. The second line contains n integers a i (1 ≤ a i ≤ 200000) — the lengths of sticks.

Output
Output a single integer — the maximal number of parallelograms that is possible to assemble.

Constraints
time limit per test 2 seconds
memory limit per test 256 megabytes
