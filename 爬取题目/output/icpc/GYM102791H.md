Input
The first line contains a single integer $n$ ($1 \le n \le 2 \cdot 10^5$) — the length of the string $s$. The second line contains string $s$ of $n$ characters. Each character is either 0 or 1.

Output
Print a single integer — the maximum number of operations you can perform.

Constraints
time limit per test 2 seconds
memory limit per test 256 megabytes
