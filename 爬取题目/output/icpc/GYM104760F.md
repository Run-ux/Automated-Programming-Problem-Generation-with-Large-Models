F. Чего-то не хватает...

Кот ученый делает ревизию на складе волшебных вещей Сказочного Патруля. Он знает, что каждого типа предметов у сказочно патруля должно быть по $S$ штук.

Кот посчитал все предметы и понял, что какого-то типа предметов всего $P$ штук, причем $P \lt S$.

Кот выяснил тип каждого из предметов на складе. Помогите Коту понять, какого типа предметов не хватает.

Input
Первая строка содержит одно целое число $N$ $(3 \le N \le 10^5)$ — суммарное количество предметов на складе. Во второй строке содержится $N$ целых чисел $t_i$ $(-2 \cdot 10^9 \le t_i \le 2 \cdot 10^9)$ — тип $i$ -го предмета на складе. Гарантируется, что существуют такие целые числа $P$ и $S$ $(1 \le P \lt S \le N)$ такие, что каждое значение $t_i$ встречается ровно $S$ раз, кроме одного значения, которое встречается ровно $P$ раз.

Output
Выведите единственное целое число — тип предмета, которого не хватает на складе.

Constraints
ограничение по времени на тест 1 секунда
ограничение по памяти на тест 256 мегабайт
