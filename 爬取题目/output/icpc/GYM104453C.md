C. Битовые операции

Антон недавно поступил в университет, и многое для него в новинку. Так, например, сегодня на программировании он изучал битовые операции, в частности, операцию побитового исключающего «или» двух чисел (xor). Данная операция существует во всех современных языках программирования, например, в языках $C\!+\!+$, $Python$ и $Java$ она обозначена как « $ ^\wedge $ », в $Pascal$ – как «xor». Пример применения операции «xor»: $10101_2 \oplus 10001_2 = 00100_2$. Можно заметить, что «xor» аналогичен сложению соответствующих битов чисел по модулю 2. У Антона есть два $N-$ значных двоичных числа (возможно, содержащие лидирующие нули). Помогите ему найти результат побитового исключающего «или» двух этих чисел, отбросив лидирующие нули.

Input
В первой строке содержится число $N$ $(1 \le N \le 10^5)$ – количество цифр в каждом из чисел. Во второй строке содержится первое двоичное число. В третьей строке содержится второе двоичное число.

Output
В единственной строке выведите двоичное число – результат исключающего «или» без лидирующих нулей.

Constraints
ограничение по времени на тест 1 секунда
ограничение по памяти на тест 256 мегабайт
