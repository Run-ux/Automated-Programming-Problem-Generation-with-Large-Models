K. XOR Clique

BaoBao has a sequence a 1, a 2, ..., a n. He would like to find a subset S of {1, 2, ..., n } such that, and | S | is maximum, where means bitwise exclusive or.

Input
There are multiple test cases. The first line of input contains an integer T, indicating the number of test cases. For each test case: The first line contains an integer n (1 ≤ n ≤ 10 5), indicating the length of the sequence. The second line contains n integers: a 1, a 2, ..., a n (1 ≤ a i ≤ 10 9), indicating the sequence. It is guaranteed that the sum of n in all cases does not exceed 10 5.

Output
For each test case, output an integer denoting the maximum size of S.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
