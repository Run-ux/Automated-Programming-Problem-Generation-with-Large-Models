Input
The first line contains an integer $n$ ($1 \leq n \leq 10^5$), indicating the number of numbers to be classified. The following $n$ lines each contain an integer $a_i$ ($1 \leq a_i \leq 10^5$), indicating the number to be classified.

Output
For each number, you must print a line with the number followed by its classification. The number and each classification should be separated by a space.
