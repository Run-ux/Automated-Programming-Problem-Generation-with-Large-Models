A. Matrix Minors

You're given an $n \times n$ matrix $A$. The minor $M_{ij}$ of $A$ in $(i,j)$ is the determinant of the matrix obtained from $A$ by deleting the $i$ -th row and the $j$ -th column. Your task is to find all minors of $A$.

Input
First line of input contains a single integer $n$ ($2 \leq n \leq 500$). The $i$ -th of the following $n$ lines contains $n$ numbers $A_{i1},\dots,A_{in}$, the $i$ -th row of $A$ ($0 \leq A_{ij} \lt 10^9+7$).

Output
Print $n$ lines, containing $n$ numbers each. The $i$ -th line should contain $M_{i1},\dots,M_{in}$, the minors of $A$. Since the resulting numbers can be very big, print them modulo $10^9+7$.

Constraints
time limit per test 3 seconds
memory limit per test 256 megabytes
