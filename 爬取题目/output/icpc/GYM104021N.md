N. Fibonacci Sequence

The Fibonacci sequence is a sequence of natural numbers, and is defined as follows:

Write a program to output the first $5$ numbers in the Fibonacci sequence.

Input
There is no input for this problem.

Output
Output $5$ integers indicating the first $5$ numbers in the Fibonacci sequence. Any two adjacent numbers in the output are separated by exactly one space and there is no extra space or symbol at the end of the line.

Constraints
time limit per test 1 second
memory limit per test 512 megabytes
