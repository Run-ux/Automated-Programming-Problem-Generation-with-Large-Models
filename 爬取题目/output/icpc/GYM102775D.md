D. Различные элементы

Эта задача была придумана авторами самой первой, а вот предысторию к ней так и не сочинили. Поэтому мы вас просим найти количество различных элементов в массиве, состоящем из N чисел a i, которые больше x и меньше y.

Input
В первой строке число N (1 ≤ N ≤ 150000) — количество элементов в массиве. Во второй строке два числа x и y ( - 2·10 9 ≤ x ≤ y ≤ 2·10 9) В третьей строке находятся элементы массива a i ( - 2·10 9 ≤ a i ≤ 2·10 9).

Output
Вывести одно число — ответ на задачу.

Constraints
ограничение по времени на тест 1 секунда
ограничение по памяти на тест 64 мегабайта
