C. Дорога

Вдоль дороги в некоторых местах висят фонари.

Фонари висят на разной высоте, некоторые фонари не работают, так что дорога освещена неравномерно — некоторые части освещаются несколькими фонарями, а некоторые вообще не освещены.

Каждый фонарь освещает некоторый интервал. Требуется определить суммарную длину освещенной части дороги.

Input
В первой строке находится одно целое $N$ ($1 \le N \le 10^4$) — количество фонарей. Каждая из следующих $N$ строк содержит через пробел два целых числа $A_i$ и $B_i$ $(-10^9 \le A \lt B \le 10^9)$ — начало и конец отрезка, освещаемого $i$ -м фонарем.

Output
Вывести одно целое число — суммарную длину освещенной части дороги.

Constraints
ограничение по времени на тест 1 секунда
ограничение по памяти на тест 256 мегабайт
