H. Heavenly Sequence

A sequence $b[1],b[2],b[3],\dots,b[n]$ is $\textbf{good}$ if and only if, $n=1$ $\textbf{or}$, for some $\textbf{non-negative integers}$ $K$ and $X$, the sequence satisfies: $$\displaystyle K \cdot \min(b[1],b[2],\dots,b[n]) + X \cdot \max(b[1],b[2],\dots,b[n]) \geq b[n]^2 + \sum_{i=1}^{n-1} b[i] \cdot b[i+1]$$

A sequence $b[1],b[2],b[3],\dots,b[n]$ is $\textbf{best}$ if and only if $\textbf{every subsequence}$ of $b[1],b[2],b[3],\dots,b[n]$ is $\textbf{good}$.

A sequence $b[1],b[2],b[3],\dots,b[n]$ is $\textbf{heavenly}$ if and only if $\textbf{every permutation}$ of $b[1],b[2],b[3],\dots,b[n]$ is $\textbf{best}$.

Given the sequence $a[1],a[2],a[3],...,a[n]$, you have to answer $n-1$ questions. The $i$ -th question is " Given integers $L$ and $R$, what is the minimum $\textbf{non-negative integer}$ $K$ that makes the sequence $a[1],a[2],a[3],\dots,a[i+1]$ $\textbf{heavenly}$, if you can choose any integer $X$ from the given range $L$ to $R$ ? "

The $\textbf{subsequence}$ of a sequence can be obtained by erasing some (possibly zero) elements from the array. (not necessary to remove consecutive elements)

Input
The first line contains integer $n$ $(2 \leq n \leq 100,000)$ — the length of the sequence The second line contains $a[1], a[2], a[3], \dots, a[n]$ $(1 \leq a[i] \leq 1,000,000)$ — the elements of the sequence The next $n-1$ lines contain integers $L$ and $R$ $(1 \leq L \leq R \leq 1,000,000)$ — the range of an integer $X$ that you can choose in each question

Output
Output $n-1$ lines, each line contains a single value — the $i$ -th value is the smallest $\textbf{non-negative integer}$ $K$ that makes the sequence $a[1],a[2],a[3],\dots,a[i+1]$ become $\textbf{heavenly}$, if you can choose any integer $X$ from the given range $L$ to $R$

Constraints
time limit per test 2 seconds
memory limit per test 512 megabytes
