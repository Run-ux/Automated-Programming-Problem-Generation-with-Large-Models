E. Old Cipher

Pikard intercepted the Borg's encryption and a set of some numbers. He remembered how such messages are encrypted.

The keys are natural numbers rearranged in a specific order:

The recorded numbers are the keys in the encryption.

You need to help Pikard correctly restore the order of the numbers for decryption.

Input
The first line contains the number $N$ $(1 \le N \le 10^5)$ — the number of numbers in the set. In the second line, there are $N$ integers $a_i$ $(1 \le a_i \le 10^9)$ — the $i$ -th number from the set, separated by spaces.

Output
Output $N$ integers separated by spaces — the numbers from the set in the order for decryption.

Constraints
time limit per test 1 second
memory limit per test 256 megabytes
