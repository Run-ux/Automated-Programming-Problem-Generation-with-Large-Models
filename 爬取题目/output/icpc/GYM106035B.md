B. Language for machines

Preprocessing of one of the languages designed for equipment management consists of the following steps:

Spaces (code 32) and tabs (code 9) are considered whitespace characters in the language.

The program always ends with an empty line.

The conversions are carried out in the specified order:

Write a program that performs preprocessing of the described type.

Input
Input contains of program consisting of at least one non-empty program line (not a comment). The program can contain only ASCII characters with codes from 32 to 127, tabs (code 9), plus line feeds in the form of a combination of characters with codes 13(CR) and 10 (LF) in the specified order. The size of the input program should not exceed 64 KB.

Output
Output a program containing at least one non-empty line, converted according to the above rules.

Constraints
time limit per test 0.25 seconds
memory limit per test 256 megabytes
