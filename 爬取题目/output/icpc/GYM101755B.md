B. Minimal Area

You are given a strictly convex polygon. Find the minimal possible area of non-degenerate triangle whose vertices are the vertices of the polygon.

Input
The first line contains a single integer n (3 ≤ n ≤ 200000) — the number of polygon vertices. Each of the next n lines contains two integers x i and y i (- 10 9 ≤ x i, y i ≤ 10 9) — the coordinates of polygon vertices. The polygon is guaranteed to be strictly convex. Vertices are given in the counterclockwise order.

Output
It is known that the area of triangle whose vertices are the integer points on the grid is either integer or half-integer. Output a single integer — the required area, multiplied by 2.

Constraints
time limit per test 2 seconds
memory limit per test 256 megabytes
