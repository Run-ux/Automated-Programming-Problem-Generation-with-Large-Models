J. Jeronimo's List

Jeronimo the bear loves numbers and he is planning to write n numbers in his notebook.

After writing the first m numbers, Jeronimo felt that he was spending a lot of time thinking new numbers, so he wrote the next n - m missing numbers as the sum modulo 3 × 10 7 of the numbers in the i - m and i - m + 1 positions for m < i ≤ n

While Jeronimo was writing, his sister Lupe arrived and asked him q questions. The i - th question consist of a number b i, Jeronimo has to say what would be the number in the position b i if all the numbers were sorted in ascending order. Jeronimo wants to answer each question as soon as possible but he spends a lot of time counting so he ask your help.

Input
The first line of the input has three integers n (3 ≤ n ≤ 3 × 10 7), m (3 ≤ m ≤ min (100, n)) and q (1 ≤ q ≤ 10000). The second line contains m numbers a 1, a 2, ..., a m, (0 ≤ a i < 3 × 10 7), The first m numbers that Jeronimo wrote. The third line contains q questions b 1, b 2, ..., b q (1 ≤ b i ≤ n)

Output
Print q lines. The i - th line must be the answer of the i - th question made by Lupe.

Constraints
time limit per test 2 seconds
memory limit per test 1024 megabytes
